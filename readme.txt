Prevent authors from uploading posts without fulfilling requirements.

== Description ==
This plugin will let you (the administrator) to prohibit any article from being published without any of the following: a thumbnail image, a title page, or a category other than Uncategorized.



== Installation ==
1. Install this plugin.
2. Use the WSR Options page under Tools to toggle which requirements to enforce upon.
